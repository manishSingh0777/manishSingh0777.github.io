It's HTML Notes

P Tag
Paragraph tag help you to write a Paragraph.
H1 Tag
H1 tag helps you write important text in bold and big size.We can use it only one time.
Nav Tag
Navigation tag helps you navigate from one page to other and back.
Ol Tag
Ordered List tag helps you write your list in a ordered way by default numbering starts from 1 but you can change it to Alphabets lower or upper case.
Img Tag
Image tag helps put your image up in your website you need specify the location of the image you want up there.Even the width and height can be adjusted.
